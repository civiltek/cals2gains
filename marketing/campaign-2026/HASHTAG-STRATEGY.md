## Content taxonomy (pilares)

### Pilar 1: Educación nutricional
**Temas:** Macros, calorías, mitos, déficit/superávit, proteína, timing, suplementos
**Formatos preferidos:** Reels (mito-busting), Carousels (guías paso a paso)
**Hashtag focus:** Set B + Set D + Set F nicho
**Frecuencia:** 2x/semana
**Objetivo:** Posicionar como autoridad, generar guardados y shares

### Pilar 2: Producto/Funciones
**Temas:** Demo escaneo, coach IA, modos, recetas, streaks, batch scan
**Formatos preferidos:** Reels (demos cortos), Stories (deep dives)
**Hashtag focus:** Set A + Set E
**Frecuencia:** 2x/semana
**Objetivo:** Conversión directa, mostrar valor del producto

### Pilar 3: Comunidad/UGC
**Temas:** Testimonios, transformaciones, "escanea mi comida", challenges
**Formatos preferidos:** Reels (compilaciones), Carousels (antes/después), Stories (repost)
**Hashtag focus:** Set A + Set C
**Frecuencia:** 1x/semana (feed) + diario (Stories)
**Objetivo:** Social proof, engagement, fidelización

### Pilar 4: Lifestyle/Tendencias
**Temas:** WIEIAD (what I eat in a day), meal prep, recetas, trending sounds
**Formatos preferidos:** Reels (trends), Stories (casual)
**Hashtag focus:** Set C + Set F nicho + trending
**Frecuencia:** 1x/semana
**Objetivo:** Alcance, viralidad, mantener relevancia

### Pilar 5: Marca/Behind the scenes
**Temas:** Equipo, proceso de desarrollo, valores, visión, actualizaciones
**Formatos preferidos:** Stories (casual), Reels (ocasional)
**Hashtag focus:** Set A + Set E
**Frecuencia:** 2-3x/mes
**Objetivo:** Humanizar la marca, construir confianza

## Hashtags prohibidos

Nunca usar:
- `#diet` `#dietculture` `#thinspo` `#fitspo` (asociados a contenido tóxico)
- `#weightloss` solo (usar `#healthyweightloss` o `#fatloss` en contexto)
- `#followforfollow` `#f4f` `#likeforlike` (spam signals)
- Hashtags banned por Instagram (revisar lista actualizada mensualmente)

## Rotación y testing

### A/B testing de hashtags
- Alternar Set B y Set C entre posts similares
- Trackear alcance por set en hoja de métricas
- Cada 2 semanas: evaluar qué combinación da más alcance no-follower
- Rotar hashtags de nicho (Set F) según tema del post

### Trending hashtags
- Revisar diariamente: TikTok Discover, Instagram Explore, Google Trends
- Si un trending hashtag encaja con nuestro contenido: usarlo en las primeras 2 horas
- Máximo 2 trending hashtags por post (no parecer oportunista)
- Crear contenido reactivo solo si es auténtico y relevante
