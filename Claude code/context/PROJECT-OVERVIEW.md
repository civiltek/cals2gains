# PROJECT-OVERVIEW.md — Qué es Cals2Gains

## En una frase
App móvil de nutrición con IA que entiende tu comida con una foto, planifica tus macros con coach IA adaptativo y soporta 5 modos nutricionales distintos.

## Propietaria
Judith — CivilTek (info@civiltek.es).

## Modelo de negocio
- **Precio**: $9.99 / mes (modelo freemium).
- **Suscripciones**: RevenueCat.
- **Plataformas**: iOS + Android.
- **Idiomas**: Español + English.
